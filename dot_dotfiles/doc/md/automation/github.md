# GITHUB

## INICIALIZAR GIT EN LA CARPETA LOCAL

```bash
# Ve al directorio
ansible@trixie:~$ cd ~/ansible-project/ansible-vampire

# Inicializa Git (si aún no lo hiciste)
ansible@debian:~/ansible-project/ansible-vampire$ git init
```

## Añade los archivos (respetando .gitignore)

```bash
ansible@debian:~/ansible-project/ansible-vampire$ git add .
```

> Git no incluirá lo que esté en `.gitignore`.
